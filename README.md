# StrengthSMP

---

## 🔷 Общая информация

**StrengthSMP** - это плагин для Paper 1.21.11, созданный для SMP-серверов. Каждый игрок получает **уникальный тип оружия** и **показатель силы**, которые вместе формируют его боевой стиль. Система создаёт асимметричный геймплей - все игроки разные, у каждого свои преимущества и слабости.

---

## ⚡ Система силы

**Сила** - это числовой показатель каждого игрока, который влияет на наносимый урон.

| Показатель | Эффект |
|---|---|
| `+1` и выше | Увеличивает урон на `сила / 1.5` за удар |
| `0` | Без бонусов и штрафов |
| `-1` и ниже | Уменьшает урон на `сила / 1.5` за удар |

**Диапазон силы:** от `-3` до `+5` (настраивается в конфиге)

### Как получить силу:
- Использовать правым кликом - добавляет `+1` к силе
- Предмет выпадает из мёртвого игрока

### Как потерять силу:
- При смерти игрок теряет `1` силу, а предмет силы выпадает на землю

---

## 🗡️ Система оружия

При первом входе на сервер игроку выдаётся **случайный тип оружия**. Оружие определяет пассивную способность и ульту игрока.

### Типы оружия:
> 🗡️ Меч · 🪓 Топор · 🔱 Трезубец · 🛡️ Щит · 🏹 Лук · ➶ Арбалет

### Смена оружия:
- Получить **предмет переброса**
- Использовать правым кликом - выдаёт случайное **другое** оружие

---

## ⚔️ Способности оружия

Каждое оружие имеет **пассивку** и **ульту**.

- **Пассивка** - активируется автоматически при выполнении условий
- **Ульта** - заряжается через пассивку, активируется командой `/ability`
- После использования ультаа начинается **кулдаун**
- Для активации ульты нужна **минимальная сила** (`+5` по умолчанию)

---

### 🗡️ Меч

#### Пассивка - Автокрит
- Наносите **3 удара подряд** полным замахом
- Следующие атаки **автоматически становятся критами** с +50% урона
- Комбо сбрасывается через **1.5 секунды** без удара

#### Ульта - Двуручный режим
- Зарядка: **5 ударов полным замахом**
- После зарядки сообщение `🗡 ГОТОВО!`
- Активация: `/ability` с мечом в руке
- **Эффект:** копия меча появляется в левой руке, все атаки наносят **x1.5 урона**
- Вокруг игрока вращаются **красные частицы**
- Длительность: **15 секунд**
- Кулдаун: **60 секунд**
- Комбо теряется через **5 секунд** без удара - появляется сообщение `🗡 Потеряно`

---

### 🪓 Топор

#### Пассивка - Оглушение
- Нанесите **5 критических ударов** подряд полным замахом
- Враг **оглушается на 1 секунду** - не может двигаться и поворачиваться
- Не работает на **доверенных игроках** (`/trust`)

#### Ульта - Накопление урона
- Зарядка: **7 критических ударов** полным замахом
- После зарядки сообщение `🪓 ГОТОВО!`
- Активация: `/ability` с топором в руке
- **Эффект:** в течение **5 секунд** все удары по врагу **не наносят урон сразу** - вместо этого урон накапливается
- По истечении времени весь накопленный урон **выпускается одним ударом**
- Кулдаун: **60 секунд**

---

### 🔱 Трезубец

#### Пассивка - Молния
- Каждые **2 удара** трезубцем призывает **молнию** в позицию врага
- Не работает на **доверенных игроках**

#### Ульта - Волна воды
- Зарядка: **3 удара** трезубцем полным замахом
- После зарядки сообщение `🔱 ГОТОВО!`
- Активация: `/ability` с трезубцем в руке
- **Эффект:** игрок **скользит вперёд** на волне воды
- Вокруг игрока в радиусе **1.8 блока** появляется вода, которая движется вместе с ним
- Все сущности рядом **отталкиваются** в стороны
- Вода не течёт во время способности
- Длительность: **5 секунд**
- Кулдаун: **60 секунд**

---

### 🛡️ Щит

#### Пассивка - Снижение урона
- Когда щит **уходит в откат**, игрок получает **на 20% меньше урона**
- Работает автоматически, не требует активации

#### Ульта - Неуязвимость
- Зарядка: **не требуется** - ульта всегда готова при достаточной силе
- Активация: `/ability`
- **Эффект:** игрок становится **полностью неуязвимым** на 15 секунд
- Атакующие **отталкиваются** от игрока при попытке ударить
- На игроке появляется **визуальный эффект** в виде защитного пузыря
- Кулдаун: **60 секунд**

---

### 🏹 Лук

#### Пассивка - Наводящаяся стрела
- Каждый **2-й выстрел** (по умолчанию) становится **наводящимся**
- Стрела автоматически **поворачивается в сторону ближайшей цели** в радиусе 12 блоков
- Не наводится на **доверенных игроков**

#### Ульта - Три луча
- Зарядка: **10 попаданий** по любым сущностям
- После зарядки сообщение `🏹 ГОТОВО!`
- Активация: `/ability` с луком в руке
- **Эффект:** выпускает **3 мощных луча** один за другим с интервалом 2 секунды
- Каждый луч:
  - Длина: **20 блоков** прямо перед игроком
  - Ширина: **1.5 блока**
  - Урон: **8 единиц здоровья** (4 сердца) по всем сущностям в зоне поражения
- Кулдаун: **60 секунд**

---

### ➶ Арбалет

#### Пассивка - Двойной урон
- Каждые **3 выстрела** следующий выстрел наносит **x2 урона**
- Особая стрела подсвечивается **эффектом свечения**
- Попавший враг получает **эффект свечения** на 7 секунд

#### Ульта - Притяжение
- Зарядка: **5 попаданий** по любым сущностям
- После зарядки сообщение `➶ ГОТОВО!`
- Активация: `/ability` с арбалетом в руке
- **Эффект:** в течение **15 секунд** все стрелы арбалета становятся **особыми**
- При попадании особой стрелы во врага:
  - Враг начинает **медленно притягиваться** к игроку в течение 3 секунд
  - Звук `ELDER_GUARDIAN_CURSE` у обоих игроков
  - Не работает на **доверенных игроках**
- Кулдаун: **60 секунд**

## 📋 Команды

| Команда | Описание | Права |
|---|---|---|
| `/strength` | Показывает вашу текущую силу и тип оружия | Все |
| `/withdraw <количество>` | Выводит силу в виде предметов в инвентарь | Все |
| `/ability` | Активирует ульта способность | Все |
| `/trust add <игрок>` | Добавить игрока в список доверенных | Все |
| `/trust remove <игрок>` | Убрать игрока из списка доверенных | Все |
| `/trust list` | Показать список доверенных игроков | Все |
| `/strengthsmp info` | Открыть GUI с информацией об оружиях | OP |
| `/strengthsmp reload` | Перезагрузить конфиг | OP |
| `/strengthsmp reroll <игрок>` | Сменить оружие игроку | OP |
| `/strengthsmp setstrength <игрок> <значение>` | Установить силу игроку | OP |
| `/strengthsmp setweapon <игрок> <оружие>` | Установить оружие игроку | OP |
| `/cooldown [игрок]` | Сбросить кулдаун способности | OP |

## ⚙️ Конфигурация

```yaml
max_strength: 5           # Максимальная сила
def_strength: 0           # Начальная сила при входе
min_strength: -3          # Минимальная сила

minimum_strength_to_activate_ability: 5   # Мин. сила для ульты
minimum_strength_to_have_passive: 0       # Мин. сила для пассивки

show_duration_in_actionbar: true    # Показывать таймер в action bar
stun_axe_duration: 1                # Длительность оглушения (сек)
sword_ultimate_multiplier: 1.5      # Множитель урона ульты меча
mace_cooldown: 30                   # Кулдаун булавы (сек, 0 = выкл)
no_strength: true                   # Запретить зелье силы
deal_less_damage: true              # Штраф к урону при отриц. силе
bow_beam_damage: 4                  # Урон луча лука (в сердцах)
bow_beam_range: 20                  # Дальность луча (блоки)
bow_beam_width: 1.5                 # Ширина луча (блоки)

# Кулдауны способностей (секунды)
sword_cooldown: 60
axe_cooldown: 60
trident_cooldown: 60
shield_cooldown: 60
bow_cooldown: 60
crossbow_cooldown: 60

# Длительности способностей (секунды)
sword_duration: 15
axe_duration: 5
trident_duration: 5
shield_duration: 15
crossbow_duration: 15
bow_beams: 3              # Количество лучей

# Требования для ульты (ударов/попаданий)
sword_ultimate: 5
axe_ultimate: 7
trident_ultimate: 3
crossbow_ultimate: 5
bow_ultimate: 10

# Требования для пассивки (ударов/попаданий)
sword_passive: 3
axe_passive: 5
trident_passive: 2
crossbow_passive: 3
bow_passive: 2
```

## 🛡️ Система доверия

Чтобы избежать случайного урона по союзникам, существует система **доверия**:

- Добавьте игрока командой `/trust add <ник>`
- Ваши способности **не будут действовать на** доверенных игроков
---

## 🔒 Особые механики

### Булава
- Имеет **кулдаун в 30 секунд** после удара (настраивается в кофниге)

### Зелье силы
- Использование зелья силы **полностью запрещено**
- При попытке применить его - зелье не действует и выводится сообщение

### Полный замах
- Большинство пассивок и ульт считают только удары с **полным замахом** (`attack cooldown = 1.0`), но это можно изменить в конфиге

### Сохранение данных
- Все данные (сила, оружие, доверенные игроки) сохраняются в файл `playerdata.yml`
